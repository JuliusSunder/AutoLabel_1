# prompt

Beschreibung: Erstelle einen reinen Textprompt (ohne Code) für einen neuen Agent/Chat den ich erstellen werde. Dieser soll folgende Anforderungen erfüllen:
- beschreibt die Anforderungen detailliert und gibt den relevanten Kontext der von den Änderungen betroffen ist
- erwähnt eventuelle Risiken/Probleme die auftreten können 
- sorgt für eine sichere Umsetzung
- lässt zwar ein bisschen Freiraum, damit der Agent die Anforderungen bestmöglich umsetzten kann, fügt aber keine unnötigen Änderungen hinzu
- wenn ich in meinen Anforderungen etwas sehr wichtiges nicht beachtet habe kannst du es gerne hinzufügen, sobald es wirklich nötig ist und keine Probleme verursacht

This command will be available in chat with /prompt
