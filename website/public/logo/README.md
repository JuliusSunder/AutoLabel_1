# Logo Instructions

Bitte speichere das AutoLabel Logo hier als `logo.png` mit transparentem Hintergrund.

**Optionen zum Entfernen des Hintergrunds:**

1. **Online Tools:**
   - https://www.remove.bg/ (automatisch)
   - https://photopea.com/ (kostenlos, wie Photoshop)

2. **Manuell mit Photopea:**
   - Öffne das Logo
   - Wähle "Magic Wand" Tool
   - Klicke auf den Hintergrund
   - Drücke Delete
   - Exportiere als PNG (transparent)

3. **Speichern:**
   - Dateiname: `logo.png`
   - Format: PNG mit transparentem Hintergrund
   - Empfohlene Größe: 200x200px oder größer (für Retina Displays)

Nach dem Speichern wird das Logo automatisch in der Navigation verwendet.
